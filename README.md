# engnovel-Scrape
this project for scraping novel web.

can take novel status such as novel description, Rating, all chapter and others


## installation
use pip to install all libraries

```bash
pip install selenium
pip install lxml
```

## you can run the program this way

```bash
python engnovel.py
```
